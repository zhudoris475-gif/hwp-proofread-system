# HWP 자동 교정 스킬

HWP 문서를 자동으로 교정하는 스킬입니다. 사용자가 "자동모드"라고 입력하면 즉시 실행됩니다.

## 실행 방법

### 1단계: 대상 파일 확인

기본 대상 폴더: `C:\Users\51906\Desktop\사전`

대상 HWP 파일:
- 【20】O 2179-2182排版页数4-金花顺-.hwp
- 【21】P 2183-2268排版页数86-金花顺.hwp
- 【大中朝 14】J 1419-1693--275--20240920.hwp
- 【大中朝 15】K 1694-1786--93--20240920.hwp
- 【大中朝 16】L 1787-1958--172--20240920-gaowm.hwp
- 【大中朝 17】M 1959-2093--135--20240920.hwp
- 【大中朝 18】N 2094-2178--85--20240920.hwp

### 2단계: 백업 실행 (최초 1회만)

```bash
cd "C:\Users\51906\Desktop\사전"
# 백업 폴더 생성
mkdir _백업_원본
# 원본 파일 백업
cp *.hwp _백업_원본/
```

### 3단계: 교정 실행

```bash
cd "C:\Users\51906\Desktop\사전"
```

**교정 스크립트 실행:**

1. **지명 변환** (중국 지명 현지음→한자음):
   ```bash
   python _china_place_convert.py
   ```

2. **왕조명 교정** (나라→조):
   ```bash
   python _dynasty_convert.py
   ```

3. **통합 규칙 교정** (띄어쓰기/붙여쓰기, 의존명사 것 등):
   ```bash
   python _integrated_convert.py
   ```

4. **전체 교정** (모든 규칙 적용):
   ```bash
   python _proofread_all.py
   ```

### 4단계: 결과 확인

교정 결과는 다음 파일에 저장됨:
- `_지명변환_결과.txt` - 지명 변환 결과
- `_왕조명_교정_보고서.txt` - 왕조명 교정 결과
- `_통합규칙_교정_보고서.txt` - 통합 규칙 교정 결과
- `*_report.txt` - 개별 파일 교정 보고서

## 주의사항

1. **백업 필수**: 실제 수정 전 반드시 `_백업_원본` 폴더에 백업
2. **한글 프로그램**: win32com 방식은 한글(HWP) 프로그램이 실행 중이면 안 됨
3. **검증**: 수정 후 텍스트 비교로 변경 사항 검증
4. **보고서**: 모든 교정 결과는 `_보고서.txt` 파일로 저장됨
