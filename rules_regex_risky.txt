# 의존명사 띄어쓰기 - 오매치 위험 패턴 (참고용)
# (\S+)와 결합시 일반명사와 겹쳐서 자동 적용 불가
# 수동 교정 또는 Ollama 판단 필요

# ── 대로 (불가) ──
# 오매치: 침대로(침대+로), 그대로(관용), 제대로(관용), 순서대로(관용)
# 교정 필요: 하는대로, 멋대로 → 하지만 일반명사와 구분 불가

# ── 뿐 (불가) ──
# 오매치: 사뿐(부사), 거뿐(형용사 활용)
# 교정 필요: 하나뿐, 거뿐 → 하지만 부사와 구분 불가

# ── 께 (불가) ──
# 오매치: 함께(한 단어) → 함 께로 분리 오류
# 교정 필요: 여러분께, 선생님께 → 하지만 함께 오매치

# ── 바 (불가) ──
# 오매치: 땅바, 손바, 곧바, 수레바, 탈바, 고무바, 카바, 오토바, 에레메이바
# 교정 필요: 할바, 갈바, 만바 (의존명사) — 일반명사와 구분 불가

# ── 것 ──
# 오매치 없는 경우: 간것, 만든것, 있는것, 이것, 그것, 저것, 어떤것, 어느것
# 보호 필요: 것이다, 것입니다, 것과, 것을, 것은, 것에
# 문제: 이것저것 (복합대명사)

# ── 수 ──
# 정확한 패턴: 할수, 낼수, 있을수, 없을수 (뒤에 있다/없다)
# 오매치: 싹수, 분수, 재수 (일반명사)

# ── 앞 ──
# 교정 필요: 문앞, 눈앞, 학교앞, 사람앞, 책상앞, 병상앞, 친구앞
# 오매치: 맨앞 (관형사+앞은 OK), 앞뒤 (복합)

# ── 밖 ──
# 교정 필요: 수밖, 문밖, 창밖, 뜻밖, 예상밖, 돈밖, 사람밖
# 오매치: 뜻밖에 (관용구로 인정)

# ── 뒤 ──
# 교정 필요: 동사뒤, 형용사뒤, 수사뒤, 이름뒤, 머리뒤
# 오매치: 앞뒤 (복합), 발뒤꿈치 (복합명사)

# ── 아래 ──
# 교정 필요: 산아래, 손아래, 하늘아래, 발아래, 문기아래
# 오매치: 우아래 (복합), 갈비아래 (해부학용어)

# ── 위 (불가) ──
# 단위, 행위, 권위, 지위, 주위, 바위, 키위, 호위, 동위, 스위스
# 따위만 교정 → 이미 (\S+)따위로 처리

# ── 안 (불가) ──
# 동안, 방안, 불안, 편안, 해안, 공안, 집안, 미안, 고안, 평안

# 교정 필요: 집안, 방안, 마당안, 교실안, 창고안
# 오매치: 동안, 불안, 편안, 해안, 공안, 미안, 고안

# ── 중 (불가) ──
# 소중, 공중, 대중, 수중, 신중, 명중, 집중, 귀중, 궁중, 열중
# 교정 필요: 도중, 행군중, 생방송중, 한밤중, 부지중, 무의식중
# 오매치: 소중, 공중, 대중, 명중, 집중, 귀중, 궁중
